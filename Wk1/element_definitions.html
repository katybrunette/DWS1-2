p
The "p" element stands for paragraph. It creates a new paragraph for your text.
<p>This is a p element example</p>
ul
The "ul" element stands for "unordered list" - it's what you use to create a bulleted list.
<ul>This is a ul element example</ul>
ol
The "ol" element stands for "ordered list" - it's what you use to create a numbered list.
<ol>This is an ol element example</ol>
li
The "li" element stands for "list item" - it tells the page that the text after the "li" element will be a list item.
<li>This is an li element example</li>
strong
The "strong" element is used to place importance on text. It makes it appear as bold.
<strong>This is a strong element example</strong>
em
The "em" element is used to place an emphasis on text. It makes it appear as italic.
<em>This is an em element example</em>
blockquote
The "blockquote" element indents text as well as creates a new paragraph. It's used with long quotations.
<blockquote>This is a blockquote element example</blockquote>
q
The "q" element inserts quotation marks into the text. It's used with shorter quotations.
<q>This is a q element example</q>
abbr
The "abbr" element is used for abbreviations.
<abbr>This is an abbr element example</abbr>
dfn
The "dfn" element is used to indicate the definition of a given term.
<dfn>This is a dfn element example</dfn>
address
The "address" element contains contact details for the page's author. It usually also has a physical address.
<address>This is an address element example</address>
ins
The "ins" element shows content that's been inserted into a document. Most of the time, it is underlined.
<ins>This is an ins element example</ins>
del
The "del" element is used to show content that has been deleted in a document. Most of the time, it is crossed out.
<del>This is a del element example</del>

The "s" element shows things that have been proven irrelevant or inaccurate. It shows this by putting a line through the text.
